hexo教學(四) 新增文章與常用指令

發表於 2020-10-23 更新於 2020-10-26 分類於 hexo教學
利用 hexo指令與 scaffolds來新增文章

新增文章

在上一篇匯入主題後，這篇我們來新增一篇文章，輸入指令後，會在 📁 _post 中看到我們新建立的檔案。
├─ 📁 source
     ├─ 📁 _post
          └─ *.md

  • 輸入 hexo new '文章標題'
  • 檔案格式為 *.md
  • 寫法為 markdown

文章與樣板 scaffolds

├─ 📁 source
     ├─ _drafts.md
     ├─ _page.md
     └─ _post.md
通常 hexo新增文章時，預設利用 📁 _source內的 _post.md新增一篇文章，其三份檔案的功能如下 :

  • _drafts.md，指令為 hexo new drafts '文章標題'，用於存放草稿且不會產生靜態檔案。
  • _page.md，指令為 hexo new page '文章標題'，新開一個導覽頁面，像是關於我、歸檔以及標籤等網站分頁。
  • _post.md，指令為 hexo new post '文章標題'，用於新增文章且會產生靜態檔案。

常見 markdown語法

Markdown語法有非常多，這邊例舉幾個常用語法。

標題類

符號 功能
# h1標題~h6標題
###### h1標題~h6標題
- + 空格 無序清單
1. + 空格 有序清單
> 引用
`` 程式碼註解
``` 程式碼區塊(上面三個下面三個)

文字類

符號 功能 實際效果
**文字** 粗體 文字
*文字* 斜體 文字
<u>文字<u> 底線 文字
~~文字~~ 刪節線 文字
<font color=#FF0000>紅色</font> 文字顏色 紅色
[連結文字](網址) 超連結文字 連結文字
![圖片文字](網址) 圖片
